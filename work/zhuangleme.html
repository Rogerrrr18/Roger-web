Social + AI Web App

zhuangleme

一款「社交话术生成器」。饭局、会议、朋友圈、路演、面试——每个场景都能开口就有料。核心不是给你一份模板，而是基于实时资讯 + 行业黑话动态生成专属话术，让你在任何社交场合说出让人眼前一亮的「内行话」。

角色 产品 + 全栈开发
技术栈 React · Python · Supabase
状态 已上线

核心场景

  • 饭局 / 聚餐 — 实时行业热点 + 黑话话术，融入圈子，建立人脉
  • 会议 / 汇报 — 专业术语包装观点，展示判断力，争取资源
  • 朋友圈 / 群聊 — 建立行业人设，获赞，展示洞察
  • 路演 / 融资 — 结构化现场发言准备，说服投资人
  • 面试 / 求职 — 展示认知深度，高情商自我介绍

技术架构

前端 React + TypeScript，Vite 构建，UI 层用 shadcn/ui。后端 Python FastAPI，Agent 路由层接入多种 LLM，Supabase 做用户体系、邀请码机制与数据持久化。

核心 Agent 是多步骤流水线：实时资讯检索 → 行业黑话注入 → 场景适配生成 → 个性化调校，SSE 流式输出，保持响应速度。冷启动用邀请码控制早期用户质量，积分机制与商业化并行。
